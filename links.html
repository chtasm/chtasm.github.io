Links

Here are some useful links!

Cycle-Ergo Allows you to find out whether or not you can properly straddle a certain motorcycle. Has almost every motorcycle ever made.
2023 SV650 Official page on Suzuki's website for the 2023 SV650.
California Motorcycle Safety Program California Motorcycle Safety Program, the perfect place to start if you live in California and want to learn how to ride a motorcycle!
Motorcycle Safety Foundation If you live outside of California, this is the website you'd use if you were interested in learning how to ride. Lots of good reading resources as well!
Used Motorcycle Evaluation Guide An extremely comprehensive guide on how to purchase a used motorcycle. Covers everything from the front tire to rear tire, everything in between, and everything that's not (history, questions to ask, etc.). Invaluable resource if you're shopping for a bike, new or used.
